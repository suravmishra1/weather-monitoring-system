# 🌦️ Weather Monitoring System

## 📌 Overview
This project monitors environmental conditions like temperature and humidity in real-time using sensors.

## ⚙️ Components Used
- Temperature Sensor (DHT11/DHT22)
- Arduino
- LCD Display (optional)

## 🔄 Working
The sensor collects environmental data and displays it in real-time.

## 🎯 Features
- Real-time monitoring  
- Accurate sensor data  
- Useful for environmental tracking  

## 🚀 Future Scope
- Add IoT dashboard  
- Send data to mobile/web
